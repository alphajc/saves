# saves
作为我个人的毕业设计项目，这是一个加密对象存储系统，我将其命名为Saves，该项目将分成多个部分。

## 功能结构
* 上传文件
* 下载文件
* 加密文件([eos-core][2])
  
  对已有的文件进行加密存储，并将其商业属性和数据特点存入数据库。以下流程：
    1. 获取缓存文件
    1. 加密
    2. 生成相关属性
    3. 放入文件库
    4. 相关数据放入数据库
    5. 清除缓存
    
    
* 解密文件([eos-core][2])

  将文件系统中的文件进行解密，并将解密后的文件放到指定位置。以下流程：
    1. 读取加密文件位置
    2. 将文件解密到指定位置
    
* 查询文件([eos-core][2])
  
  检索数据库中的文件信息，并返回
